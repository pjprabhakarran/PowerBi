The projects hosted here kick starts from very basic dashboard creation to data cleaning, schema mapping etc.

About the Project:

Sales_codebase is a sales transaction reporting dashboard, this aims to resolve the issue of sales director of a company from the manual updates from the team, company is spread across India it was a hectic activity for the sales director earlier to collate the data and take timely action.

Key Acitivity of this project:

- Data Extraction from the MySql data based into Power BI
- Data cleaning (Null removal, duplicate removal)
- Data Transformation (USD to INR, Date Format change)
- Identify key metrics for the sales manager
- Create interative dash board with the identified key metrics
- Giving it an visually informative appeal
